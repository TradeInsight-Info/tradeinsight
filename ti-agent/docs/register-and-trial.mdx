`ti` is bring-your-own-model, so the fastest way to start is to run it with your
own provider — [GitHub Copilot](/docs/ti-agent/github-copilot),
[ChatGPT (Codex)](/docs/ti-agent/codex-subscription), or an
[Anthropic API key](/docs/ti-agent/anthropic-api). Signing in to TradeInsight
adds the finance features on top — the full insight dataset, live market data,
and profile sync.

**TradeInsight-provided models are a work in progress** and not available yet.
If you'd like to run `ti` on models we host — no key of your own — we'd love to
hear from you: [get in touch](https://tradeinsight.info/contact) to register your
interest, and start the free trial below.

## 1. Create your account

Go straight to **[app.tradeinsight.info/upgrade/ai-max](https://app.tradeinsight.info/upgrade/ai-max)** —
the Finance Agent page. If you don't have an account yet you'll sign in first:
it's passwordless — enter your email, paste the one-time code we send, and you
land back on the Finance Agent page. That same account is what the CLI signs in
to.

## 2. Start the free trial

On the Finance Agent page, click **Start free AI trial**.

- Free for **7 days**, no credit card, one per account.

Meanwhile, connect your own provider (below) so you're up and working right now.

## 3. Install the CLI

```bash
bun install -g @tradeinsight-info/ti
# or: npm install -g @tradeinsight-info/ti
```

Requires Node.js **22.19** or newer. See [Getting started](/docs/ti-agent/getting-started)
for the full install notes.

## 4. Sign in from the CLI

Launch it and sign in to TradeInsight:

```bash
ti
```

At the prompt:

1. **`/login-ti`** — signs in to TradeInsight in your browser. It runs
   automatically on first launch; use it any time to refresh your session. Your
   TradeInsight token is the only thing sent to TradeInsight — LLM provider keys
   never leave your machine.
2. **`/login`** — connect your own model provider (Copilot, Codex, Anthropic, and
   more), then **`/model`** to pick which one this session runs on.

That's it — you're set up and can start asking `ti` to pull data, sketch a
strategy, or run a backtest.

## When the trial ends

After 7 days the trial closes. We're working towards a paid subscription for
TradeInsight-provided models. Need more trial time? [Get in touch](https://tradeinsight.info/contact),
tell us what you're building and why you'd like more, and we'll sort it out.

## Troubleshooting

- **Sign-in didn't open a browser** — `/login-ti` prints a URL; open it manually,
  finish signing in, and the CLI continues automatically.
- **No model responding** — you haven't connected a provider yet. Run `/login`,
  pick your provider, then `/model`. See [Choose your model](/docs/ti-agent/getting-started#choose-your-model).
