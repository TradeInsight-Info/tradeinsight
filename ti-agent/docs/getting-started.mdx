`ti` is TradeInsight's finance & trading CLI agent — an AI coding agent tuned for
market research, backtesting, and portfolio work, published as
`@tradeinsight-info/ti`.

## Requirements

- Node.js **22.19** or newer.

## Install

```bash
npm install -g @tradeinsight-info/ti
```

## Run

```bash
ti
```

This starts an interactive session. Ask it to pull market data, sketch a
strategy, or run a backtest, and it works through the task step by step.

## What it can do

- Research congressional and corporate-insider trades.
- Fetch OHLCV price history and market tickers.
- Draft and backtest systematic strategies with honest in-sample / out-of-sample splits.

See the [changelog](/docs/ti-agent/changelog) for what's new in each release.
