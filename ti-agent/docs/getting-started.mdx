`ti` is TradeInsight's finance & trading CLI agent — think of it as Claude Code
for quants and traders. It researches the market, backtests strategies, and works
on portfolios in plain language, published as `@tradeinsight-info/ti`.

## Requirements

- Node.js **22.19** or newer.

## Install

```bash
bun install -g @tradeinsight-info/ti
# or: npm install -g @tradeinsight-info/ti
```

## Run

```bash
ti
```

This starts an interactive session. Ask it to pull market data, sketch a
strategy, or run a backtest, and it works through the task step by step.

## First run — sign in

On first launch `ti` opens a TradeInsight sign-in in your browser. You can keep
working while it's open, and declining is fine — but signing in enables profile
sync and the full TradeInsight finance dataset.

Two quick steps at the prompt:

1. **`/login-ti`** — sign in to TradeInsight (opens the browser). Runs
   automatically on first start; use it any time to sign in again.
2. **`/login`** — connect the model you want to run `ti` with (see below).

Then pick a model with **`/model`**.

Your TradeInsight sign-in and your model credentials are separate. LLM provider
keys stay on your machine — only your TradeInsight token is sent to TradeInsight.

## Choose your model

`ti` is bring-your-own-model. `/login` connects your own provider — supported:
OpenAI Codex, OpenCode Zen, GitHub Copilot, Anthropic API, OpenAI API, AWS
Bedrock, and more. Use `/model` to pick which one a session runs on.

**TradeInsight-provided models** (models we host, no key of your own required)
are on the way but not available yet — [tell us if you'd like them](https://tradeinsight.info/contact).
For now, connect your own provider above.

## Set up your profile (optional)

Run **`/profile`** for a short interview about how you invest — trader vs.
investor, risk tolerance, horizon, instruments. `ti` tailors its analysis to your
answers and syncs the profile across your sessions and the web dashboard.

## What it can do

- **Backtest strategies and portfolios** — risk-parity, min-vol, vol-target,
  momentum, DCA, and leverage — with honest in-sample / out-of-sample splits.
- **Verify market claims** — strategies, factors, and calendar/seasonal cycles
  (sell-in-May, momentum, the BTC halving cycle) tested on real data rather than
  taken as folklore.
- **Hedge downside risk** — size and backtest beta hedges, diversifiers, and
  exposure trims.
- **Price options locally** — Black-Scholes value, the Greeks, implied
  volatility, and multi-leg payoffs (covered calls, spreads, straddles).
- **Reason about fixed income** — bonds, duration and rate risk, and picking a
  bond ETF for a goal.
- **Analyze companies** through a value-investing (Buffett/Munger) lens, and
  generate a self-contained stock-analysis report.
- **Pull market data** — daily OHLCV price history (cached locally), quotes, and
  recent congressional & corporate-insider trades by ticker.

## Useful commands

| Command | What it does |
|---------|--------------|
| `/login-ti` | Sign in to TradeInsight |
| `/login` | Connect your LLM provider |
| `/model` | Choose the model for this session |
| `/profile` | Set up your trading profile |
| `/logout-all` | Sign out of TradeInsight and every connected provider |
| `/new`, `/resume` | Start a new session or resume a previous one |

Type `/` at the prompt to see every available command.

## Good to know

- `ti` **never places orders or connects to a broker** — you stay in control.
- Market data is **exchange-delayed** (~15 min for US quotes), so `ti` is built
  for research and swing/positional analysis, not intraday trading.
