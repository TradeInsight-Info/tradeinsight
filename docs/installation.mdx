The Data API is a plain REST endpoint — no SDK required. You can use it from any HTTP client.

## 1. Get an API key

Request a free beta key at [beta.tradeinsight.info](https://beta.tradeinsight.info). Your key arrives in your inbox immediately — no credit card needed.

## 2. Confirm it works

```bash
curl -H "Authorization: Bearer $TRADEINSIGHT_API_KEY" \
  https://api.tradeinsight.info/trading-data/v1/health
```

You should see:

```json
{ "status": "ok" }
```
